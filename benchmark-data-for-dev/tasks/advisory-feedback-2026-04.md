# Замечания от Advisory (Iza) — Апрель 2026

## Критические (влияют на UX и первое впечатление)

### 1. Мигающий текст "Check your business..."
- **Страница:** Главная
- **Проблема:** Надпись сильно мигает, вызывает визуальный дискомфорт
- **Решение:** Убрать или смягчить анимацию мигания
- **Приоритет:** Высокий

### 2. Неправильная позиция скролла на странице выбора параметров
- **Страница:** Страница с выбором страны/уровня отчётности
- **Проблема:** Страница открывается на уровне "основной страны деятельности", а не сверху. Пользователь может не заметить поле выбора отрасли и оставить значение по умолчанию (строительная отрасль)
- **Решение:** При загрузке страницы скроллить к началу формы
- **Приоритет:** Высокий

### 3. Неоднозначность в выборе "Кто вы в этом процессе"
- **Страница:** Форма выбора роли
- **Проблема:** Если компания одновременно является МСП и поставщиком — непонятно какую опцию выбирать. Нет ясности, что важнее для отчёта
- **Решение:** Добавить пояснение или возможность множественного выбора
- **Приоритет:** Высокий

## Средние (качество отчёта)

### 4. Смешение языков в отчёте
- **Где:** Сгенерированный отчёт
- **Проблема:** Часть отчёта генерируется на польском, часть на английском
- **Решение:** Сделать отчёт полностью на одном языке. Добавить выбор языка (польский/английский)
- **Приоритет:** Средний

### 5. Опечатка "ZAGRAŻENIA"
- **Где:** Страница 2 отчёта, заголовок экологических угроз
- **Проблема:** Написано "ZAGRAŻENIA" вместо "ZAGROŻENIA"
- **Решение:** Исправить опечатку
- **Приоритет:** Средний

### 6. Одинаковое описание оранжевого столбца
- **Где:** Раздел операционного воздействия в отчёте
- **Проблема:** Оранжевый столбец описан одинаково для всех областей E, S и G
- **Решение:** Сделать описания уникальными для каждой области
- **Приоритет:** Средний

## Низкие (улучшение UX)

### 7. Не работает прокрутка отчёта с клавиатуры
- **Где:** Просмотр отчёта
- **Проблема:** Отчёт можно прокручивать только боковым скроллбаром, стрелки клавиатуры не работают
- **Решение:** Добавить поддержку навигации с клавиатуры (focus на контейнер, tabindex)
- **Приоритет:** Низкий
